# funk-lib

Functional JavaScript Utilities


#### Commands

`$ npm run <command>`

| command      | description   |
|--------------|---------------|
| `test`       | Run tests *(source + CF template)* |
| `test:watch` | Run tests on change |
| `cover`      | Run test coverage |
| `lint`       | Lint the source |
| `verify`     | Verify linting, tests, coverage |
